HIPI Tools and Example Programs

The HIPI distribution includes a number of useful tools and example programs that are a great way to learn how to create your own HIPI programs.

hibimport

This is a simple tool and a good place to start exploring HIPI. hibimport creates a HipiImageBundle (HIB) from a folder of images on your local file system. A HIB is the key input file to the HIPI framework and represents a collection of images stored on the Hadoop Distributed File System (HDFS).

DumpHib

This example program is a good introduction to the HIPI API. It is a full MapReduce/HIPI program that collects basic information about the images in a HIB. It does this using multiple parallel map tasks (one mapper for each image in the HIB) and writes this information to a text file on the HDFS in a single reduce task. (You can use the hibimport tool or the Distributed Downloader example program to create a HIB.)

DistributedDownloader

This is a more advanced MapReduce/HIPI program that allows creating a HipiImageBundle (HIB) by performing a distributed download of a list of images located on the Internet. This program highlights some of the more subtle parts of the Hadoop framework and will be a valuable tool for setting up your Hadoop cluster with a large image collection.

JpegFromHib

This is a MapReduce/HIPI program that extracts the images within a HipiImageBundle (HIB) as individual files. This program illustrates many important features of the HIPI API (e.g., how to process the images in a HIB according to the MapReduce parallel programming model). It is also a useful tool to verify that a HIB has been properly created.

CreateSequenceFile

This is a MapReduce/HIPI program that generates a Hadoop Sequence File (a flat binary file of key/value pairs) from a HIB. It demonstrates how to use the JPEGImageUtil class that is part of HIPI and shows how to provice access to a set of images in another format. This program was used for some performance experiments.

The Principle Components of Patches

This is a MapReduce/HIPI program that implements the famous experiment described in the paper The Principal Components of Natural Images, written by Hancock et al. in 1992. This program computes the principal components of natural image patches (eigenvectors of the covariance matrix computed over a large set of small image patches). Although the original paper only used 15 grayscale images, HIPI makes it easy to perform this analysis over much larger image sets. This is a more advanced program and a good starting point for building more complex distributed image processing and computer vision MapReduce/HIPI programs.

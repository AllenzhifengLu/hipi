HIPI Example Programs

The Distributed Downloader

This is the first example you will probably want to inspect because it will create a HipiImageBundle that you can use to run the other examples or create your own programs. Although the details of this program are rather intricate, they relate primarily to Hadoop and not to HIPI so they can be safely ignored if need be.

DumpHIB

This example is a good introduction to the HIPI API. It uses a user-defined Mapper to output information about the images in a HipiImageBundle (such as one created from the Distributed Downloader).

JpegFromHib

This example will generate a set of jpeg files from an input HipiImageBundle (HIB). It both illustrates how to parse data from a HIB, and provides visual confirmation that a hib is storing images as expected.

CreateSequenceFile

This example will generate a sequence file from an input hib. It provides example usage of the JPEGImageUtil class, and it provides access to the image data as a SequenceFile, which may be useful in other applications of HIPI.

The Principle Components of Patches

This is a more complicated example program that illustrates HIPI's effectiveness at computing the Principle Components of natural image patches in a distributed environment.
